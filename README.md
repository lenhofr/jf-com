# jf-com

## Run locally

This repoâ€™s web app lives in `frontend/` and is built with Vite + React.

### Prerequisites

- Node.js (18+ recommended)
- npm

### Install

```bash
cd frontend
npm ci
```

### Start dev server

```bash
cd frontend
npm run dev
```

Vite will print the local URL (typically `http://127.0.0.1:5173/`).

### Production build

```bash
cd frontend
npm run build
```

### Preview the production build

```bash
cd frontend
npm run preview
```
